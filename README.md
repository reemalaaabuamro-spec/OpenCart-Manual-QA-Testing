# OpenCart Manual QA Testing Project

## Project Overview
This repository contains manual test cases for the OpenCart Demo Website created using Jira and Zephyr Essential.

## Tools Used
- Jira
- Zephyr Essential
- GitHub

## Modules Tested
- Register
- Login
- Forgot Password
- My Account
- Address Book
- Shopping Cart
- Checkout
- Search
- Product
- Currency
- Newsletter
- Returns
- Logout

## Total Test Cases
104 Manual Test Cases

## Testing Techniques
- Positive Testing
- Negative Testing
- Boundary Value Analysis (BVA)
- Validation Testing
